2️⃣ QUERY SQL + BINDING UI (PRONTE)
📊 STATISTICHE
Copia codice
Sql
SELECT
    COUNT(*) AS total_bets,
    SUM(CASE WHEN profit > 0 THEN 1 ELSE 0 END) AS wins,
    SUM(profit) AS total_profit,
    ROUND(AVG(profit), 2) AS avg_profit
FROM bet_history
WHERE settled_ts >= datetime('now', '-24 hours');
👉 UI:
label KPI
progress bar winrate
💰 STORICO P&L
Copia codice
Sql
SELECT
    settled_ts,
    market_id,
    side,
    stake,
    profit
FROM bet_history
ORDER BY settled_ts DESC
LIMIT 100;
👉 UI:
table
sparkline / equity curve
📡 AUDIT TELEGRAM

❗ LOG ERRORI

⚡ PERFORMANCE
Copia codice
Sql
SELECT
    ts,
    api_calls_min,
    loop_latency_ms,
    throttle_state
FROM performance_metrics
WHERE ts >= datetime('now', '-1 hour')
ORDER BY ts ASC;
👉 UI:
line chart
state badge (IDLE/NORMAL/HIGH)
