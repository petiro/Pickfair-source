1️⃣ DUTCHING BACK + LAY MISTO (UNIFORM PROFIT)

Caso reale:

alcune selezioni BACK

altre LAY

profitto identico qualunque outcome

📐 Formula concettuale

Profitto per outcome k:

profit_k =
Σ (back_stake_i * (odds_i - 1))  se i == k
- Σ (back_stake_j)               se j != k
+ Σ (lay_stake_j)                se j != k
- liability_k                    se lay su k


Obiettivo:

profit_1 = profit_2 = ... = profit_N = P


👉 Sistema lineare → risolvibile via algebra

✅ Codice Python (BACK + LAY misto)
import numpy as np

def mixed_dutching(odds, sides, target_profit, commission=0.05):
    """
    odds: list[float]
    sides: list["BACK" | "LAY"]
    """

    n = len(odds)
    A = []
    b = []

    for k in range(n):
        row = []
        for i in range(n):
            o = odds[i]
            if sides[i] == "BACK":
                if i == k:
                    row.append(o - 1)
                else:
                    row.append(-1)
            else:  # LAY
                if i == k:
                    row.append(-(o - 1))
                else:
                    row.append(1)
        A.append(row)
        b.append(target_profit)

    stakes = np.linalg.solve(np.array(A), np.array(b))
    stakes = np.maximum(stakes, 0)

    profit = target_profit * (1 - commission)

    return [
        {
            "side": sides[i],
            "odds": odds[i],
            "stake": round(stakes[i], 2),
            "profit": round(profit, 2)
        }
        for i in range(n)
    ]

def multi_market_swap(markets, target_profit):
    """
    markets = [
      {"market": "MO", "odds": 2.1, "side": "BACK"},
      {"market": "OU25", "odds": 1.9, "side": "LAY"}
    ]
    """

    exposure = 0
    for m in markets:
        if m["side"] == "BACK":
            exposure += target_profit / (m["odds"] - 1)
        else:
            exposure += target_profit / m["odds"]

    return round(exposure, 2)

📐 Formula Cashout Exchange

BACK:

cashout = (lay_odds / back_odds) * back_stake


LAY:

cashout = (back_odds / lay_odds) * lay_stake

✅ Codice Cashout Reale
def dynamic_cashout(orders, live_odds, commission=0.05):
    profit = 0

    for o in orders:
        if o["side"] == "BACK":
            profit += (o["odds"] - 1) * o["stake"]
            hedge = (o["odds"] / live_odds[o["selection"]]) * o["stake"]
            profit -= hedge
        else:
            profit += o["stake"]
            hedge = (live_odds[o["selection"]] / o["odds"]) * o["stake"]
            profit -= hedge * (live_odds[o["selection"]] - 1)

    if profit > 0:
        profit *= (1 - commission)

    return round(profit, 2)
